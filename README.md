# Rule-Based Health Diagnosis Expert System

## About the Project

This is a simple Rule-Based Health Diagnosis Expert System developed using Python. The project asks the user about different symptoms and suggests a possible disease based on predefined if-then rules.

The main objective of this project is to understand how expert systems work using forward chaining and rule-based reasoning.

## Features

- Accepts symptoms from the user.
- Stores the symptoms as facts.
- Uses if-then rules to diagnose diseases.
- Displays inference steps.
- Diagnoses Common Cold, Flu, and Food Poisoning.

## Technologies Used

- Python 3
- Visual Studio Code
- Git & GitHub

## How to Run

1. Clone or download this repository.
2. Open the project in Visual Studio Code.
3. Open the terminal.
4. Run the following command:
   
   py expertsystem.py
   
5. Answer the questions by entering *yes* or *no*.
6. View the inference steps and final diagnosis.
